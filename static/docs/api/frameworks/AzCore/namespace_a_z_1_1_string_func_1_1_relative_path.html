bool AZ::StringFunc::RelativePath::IsValid ( const char *  in)

IsValid.

Returns if this string has the requirements to be a relative path ! makes sure it has no AZ_FILESYSTEM_INVALID_CHARACTERS ! makes sure all AZ_WRONG_FILESYSTEM_SEPARATOR's are replaced with AZ_CORRECT_FILESYSTEM_SEPARATOR's ! fails if it HasDrive() ! make sure it has an ending AZ_CORRECT_FILESYSTEM_SEPARATOR ! if AZ_FILENAME_ALLOW_SPACES is not set then it makes sure it doesn't have any spaces Example: Is the AZStd::string a valid root StringFunc::Root::IsValid(a="p4\\game\\") == true StringFunc::Root::IsValid(a="p4/game/") == false; Wrong separators StringFunc::Root::IsValid(a="\\p4\\game\\") == false; Starts with a separator StringFunc::Root::IsValid(a="C:\\p4\\game") == false; Has a drive

Normalize.

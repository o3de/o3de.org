Parallel Processing

Parallel processing is supported as much as possible on each platform. It has lot's of limitations, please refer to each primitive for more information. We are conforming with C++14 but of course have have limited support.

  • AZStd::mutex (a.k.a critical_section)
  • AZStd::recursive_mutex (a.k.a critical_section)
  • AZStd::semaphore
  • AZStd::thread
  • AZStd::condition_variable
  • AZStd::lock
  • ParallelContainers

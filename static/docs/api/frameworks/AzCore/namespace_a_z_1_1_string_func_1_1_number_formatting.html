GroupDigits.

Modifies the string representation of a number to add group separators, typically commas in thousands, e.g. 123456789.00 becomes 123,456,789.00

Parameters
buffer- The buffer containing the number which is to be modified in place
bufferSize- The length of the buffer in bytes
decimalPosHint- Optional position where the decimal point (or end of the number if there is no decimal) is located, will improve performance if supplied
digitSeparator- Grouping separator to use (default is comma ',')
decimalSeparator- Decimal separator to use (default is period '.')
groupingSize- Number of digits to group together (default is 3, i.e. thousands)
firstGroupingSize- If > 0, an alternative grouping size to use for the first group (some languages use this, e.g. Hindi groups 12,34,56,789.00)
Returns
The length of the string in the buffer (including terminating null byte) after modifications

Detailed Description

Processes a driller driller and dispatches the data based on the the DrillerHandlerParser (handlers) and their ability to handle specific tags. If a tag is NOT found as a child of the current one it will display a warning with the tag name (useless it's allowed by DrillerHandlerParser::IsWarnOnUnsupportedTags) and process the stream is a safe manner by skipping all the data and tags we can't handle.

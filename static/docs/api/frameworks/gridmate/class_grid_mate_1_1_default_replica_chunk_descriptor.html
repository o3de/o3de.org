Detailed Description

template<typename ReplicaChunkType, ZoneMask mask = ZoneMask_All>
class GridMate::DefaultReplicaChunkDescriptor< ReplicaChunkType, mask >

DefaultReplicaChunkDescriptor provides a common implementation for chunk descriptors. It can be used for chunk types that do not use ctor data and have no special construction/destruction requirements.

Detailed Description

template<typename Type>
class GridMate::Marshaler< Type, typename AZStd::Utils::enable_if_c< IsFundamentalMarshalType< Type >::Value >::type >

Fundamental type marshaler. All fundamental types except bool are written here. They are endian swapped if necessary, and written raw directly to the buffer.

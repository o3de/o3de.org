Detailed Description

Description - configure the system allocator. By default we will allocate system memory using system calls. You can provide arenas (spaces) with pre-allocated memory, and use the flag to specify which arena you want to allocate from. You are also allowed to supply IAllocatorSchema, but if you do so you will need to take care of all allocations, we will not use the default HeapSchema. HeapSchema::Descriptor

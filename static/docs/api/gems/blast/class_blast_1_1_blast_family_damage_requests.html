Detailed Description

Bus that handles damage being dealt to the actors in the family. For convenience family connects on its own id as well as all entities that represent blast actors. If the damage is applied on family id then it's applied to all appropriate actors (which might be limited to just that actor, e. g. in StressDamage). If the damage is applied on actor id then it's only applied to that actor.

virtual void Blast::BlastFamilyDamageRequests::DestroyActor ( )

Destroys actor. This is not similar to damage, because the actor is not split into child actors, but rather just removed from simulation. This method still triggers notifications. Calling this on a family id will destroy all of its actors.

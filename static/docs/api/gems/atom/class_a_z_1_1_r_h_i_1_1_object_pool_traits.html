Detailed Description

Inherits from object collector traits since they share types.

Base traits class for templatizing the ObjectPool. You can customize the object pool by overriding traits in this class.

For example: class MyObjectPoolTraits : public RHI::ObjectPoolTraits { public: // Enable locking for thread-safe usage. using MutexType = AZStd::mutex;

// Specify a type (the type must derive from Object or match its interface). using ObjectType = MyObject;

// Specify a factory class which will manage creation / deletion of objects in the pool. using ObjectFactoryType = MyObjectFactory; };

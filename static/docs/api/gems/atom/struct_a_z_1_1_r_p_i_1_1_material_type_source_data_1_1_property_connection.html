int32_t AZ::RPI::MaterialTypeSourceData::PropertyConnection::m_shaderIndex = -1

For m_type==ShaderOption, this is either the index of a specific shader in m_shaderCollection, or -1 which means every shader in m_shaderCollection. For m_type==ShaderInput, this field is not used.

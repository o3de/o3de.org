Detailed Description

Hook used in ROS2 Gem Robot Importer to create ROS2 components based on SDFormat model description. It implements the parameters mapping between the SDFormat model plugin and the particular O3DE component. It should be registered as a serialization attribute in the component's reflection to make the it visible in the SDFormat model parser.

Detailed Description

This component marks a reference frame for ROS 2 ecosystem. It serves as sensor data frame of reference and is responsible, through ROS2Transform, for publishing ros2 static and dynamic transforms (/tf_static, /tf). It also facilitates namespace handling. An entity can only have a single ROS2Frame on each level. Many ROS 2 Components require this component.

Note
A robot should have this component on every level of entity hierarchy (for each joint, fixed or dynamic)

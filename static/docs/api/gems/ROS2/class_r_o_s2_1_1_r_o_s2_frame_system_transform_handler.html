Detailed Description

Handler class for the ROS2FrameEditorComponent. It watches for changes in the entity tree and notifies about moves. Used by the ROS2FrameSystemComponent, to track changes in the entity tree. It notifies the ROS2FrameSystemComponent and calls the appropriate functions to update the changes.

float AZ::Render::RenderActorSettings::m_wireframeScale = 0.1f

Use the vertex normals to scale the wireframe a bit to avoid Z-fighting when rendering. Scale the normal by the m_wireframeScale to push the wireframe a bit above the solide mesh rendering. Additionally the character bounds will be taken into account, so this is a relative-to the character size value.

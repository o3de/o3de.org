Detailed Description

Each box drawn through the AuxGeom draw interface is stored in the scene data as an instance of this struct Objects can either be Shapes or Boxes. Boxes are kept separate because they do not have LODs so they have a different processing path. It also saves the memory of storing a shape type for boxes. Keeping them in a separate list also makes instancing possible.

◆ m_deviceStreamBufferViews

AZStd::unordered_map<int, AZStd::vector<StreamBufferView> > AZ::RHI::MultiDeviceDrawPacketBuilder::MultiDeviceDrawRequest::m_deviceStreamBufferViews

A map of all device-specific StreamBufferViews, indexed by the device index This additional cache is needed since device-specific StreamBufferViews are returned as objects and the device-specific DrawItem holds a pointer to it.

◆ m_drawFilterMask

DrawFilterMask AZ::RHI::MultiDeviceDrawPacketBuilder::MultiDeviceDrawRequest::m_drawFilterMask = DrawFilterMaskDefaultValue

Mask for filtering the draw item into specific render pipelines. We use a mask because the same item could be reused in multiple pipelines. For example, a simple depth pre-pass could be present in multiple pipelines.

◆ SetFileTypeFilters()

void AtomToolsFramework::AtomToolsAssetBrowser::SetFileTypeFilters ( const FileTypeFilterVec & fileTypeFilters)

Set up ext based file filters for the asset browser. The filters can be set using the options menu on the search bar. This is intended to be used instead of asset type filters because some tools primarily work with source files that do not have registered asset type info.

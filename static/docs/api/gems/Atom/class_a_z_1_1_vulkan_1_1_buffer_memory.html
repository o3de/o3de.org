Detailed Description

A BufferMemory represents a Vulkan's Buffer that will be sub allocated into one or multiple RHI Buffer Resources. Each of the RHI Buffers reference a section of the Vulkan Buffer. The Vulkan Buffer sits on top of a Vulkan Memory Region. When it is removed, the native buffer will be freed.

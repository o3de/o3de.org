Detailed Description

System component in charge of firing the event that tells the Factory Manager that all available factories have registered. In order to do this it depends on the service provided by the platform factories implementation. This guarantees that it will always be activated after the platform factories. It also provides the RHI component service so other components can activate after the RHI is ready.

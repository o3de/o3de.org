Detailed Description

The MultiDeviceIndirectBufferSignature is an implementation object that represents the signature of the commands contained in an Indirect Buffer. Indirect Buffers hold the commands that will be used for doing Indirect Rendering.

It also exposes implementation dependent offsets for the commands in a layout. This information is useful when writing commands into a buffer.

Detailed Description

template<typename ObjectType>
class AZ::RHI::ObjectFactoryBase< ObjectType >

Base class for managing creation / deletion of objects in the ObjectPool. When creating an object pool type, the user can specify a derived variant of RHI::Object, and the pool will internally manage objects using that factory.

Example Usage: class CommandListFactory : public RHI::ObjectFactoryBase<CommandList> { public: // Override descriptor struct with your own. struct Descriptor { // Command queue type, device handle. };

void Init(const Descriptor& descriptor);

// Only override the methods you care about. CommandList* CreateObject();

private: Descriptor m_descriptor; // Store data here to inform CreateObject };

Detailed Description

MultiDeviceDrawPacket is a multi-device class that holds a map of device-specific DrawPackets as well as a vector of MultiDeviceDrawItems, corresponding SortKeys, DrawListTags and DrawListMasks. A MultiDeviceDrawPacket is only intened to be contructed via the MultiDeviceDrawPacketBuilder. Individual device-specific DrawPackets are allocated as packed data structures, referenced via RHI::Ptrs in a map, indexed by the device-index.

virtual AZ::s32 AtomToolsFramework::AtomToolsMainMenuRequests::GetMainMenuPriority ( ) const
inlinevirtual

Override this function to specify the order in which menu actions are added and updated relative to other handlers. Handlers specifying a lower number are processed first. The common use case is that we want common, application-wide main menu actions to appear at the top of menus and document/view specific actions to be added after those.

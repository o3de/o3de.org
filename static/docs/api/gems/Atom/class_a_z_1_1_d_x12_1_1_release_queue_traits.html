Detailed Description

This is a deferred-release queue for ID3D12Object instances. Any DX12 object that needs to be released on the CPU timeline should be queued here to ensure that a reference is held until the GPU has flushed the the last frame using it.

Each device has an object queue, and will synchronize its collect latency to match the maximum number of frames allowed on the device.

Detailed Description

Component responsible for building Atom's AzModel from SceneAPI input.

The current strategy in this builder is to merge data as much as possible while keeping streams separate. This may change for different platforms in the future.

Currently Meshes are merged as much as possible at the ModelLod level. An input SceneAPI Model with 3 meshes in an LOD are going to get merged into stream buffers owned by the ModelLod. The Meshes in the ModelLod are then just going to act as views into the ModelLod's buffers with an associated Material.

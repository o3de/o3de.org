Detailed Description

MultiDeviceRayTracingTlasDescriptor

The Build() operation in the descriptor allows the TLAS to be initialized using the following pattern:

RHI::MultiDeviceRayTracingTlasDescriptor descriptor; descriptor.Build() ->Instance() ->InstanceID(0) ->HitGroupIndex(0) ->Blas(blas1) ->Transform(transform1) ->Instance() ->InstanceID(1) ->HitGroupIndex(1) ->Blas(blas2) ->Transform(transform2) ;

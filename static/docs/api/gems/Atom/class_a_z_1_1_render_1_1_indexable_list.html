Detailed Description

template<typename T>
class AZ::Render::IndexableList< T >

This container class implements a doubly linked list using a vector as the underlying data type. This allows indexing via the [] operator. Insertion or removal of items will not invalidate the indices of existing items Insertion and removal times are O(1), reserve is O(n) See Intro to Algorithms, section 10.3: A single array representation of objects

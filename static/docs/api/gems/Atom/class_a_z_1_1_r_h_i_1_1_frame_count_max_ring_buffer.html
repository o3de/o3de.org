Detailed Description

template<typename T>
class AZ::RHI::FrameCountMaxRingBuffer< T >

A class which manages a FrameCountMax number of objects and manages them in a ring buffer structure, meaning that whenever an element needs to be updated, the current element index is incremented (mod FrameCountMax), which leaves the other elements unchanged, which is necessary for some resources if the GPU and CPU are not in sync.

Detailed Description

This execution context operates at a high level, and is not specific to a particular material pipeline. It can read material property values. It can set internal material property values (to pass data to pipeline-specific functors which use PipelineRuntimeContext). It can configure the Material ShaderResourceGroup because there is one for the entire material, it's not specific to a material pipeline or particular shader. It can configure shaders that are not specific to a particular material pipeline (i.e. the MaterialPipelineNone ShaderCollection). It can set shader option values (Note this does impact the material-pipeline-specific shaders in order to automatically propagate the values to all shaders in the material).

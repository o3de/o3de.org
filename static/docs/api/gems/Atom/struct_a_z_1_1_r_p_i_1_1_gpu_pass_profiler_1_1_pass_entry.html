Detailed Description

Intermediate data that represents the structure of a Pass within the FrameGraph. A tree structure will be created from these entries that mimics the pass' structure. By default, all entries have a parent<-child reference, but only entries that pass the filter will also hold a parent->child reference.

void AZ::RPI::GpuPassProfiler::PassEntry::LinkChild ( PassEntry *  childEntry)

Links the child TimestampEntry to the parent's, and sets the dirty flag for both the parent and child entry. Calling this method will effectively add a parent->child reference for this instance, and all parent entries leading up to this entry from the root entry.

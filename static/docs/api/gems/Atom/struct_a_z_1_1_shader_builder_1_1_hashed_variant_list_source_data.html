Detailed Description

This is a simple data structure that represents a .hashedvariantlist file. Users are not expected to create .hashedvariantlist files. These files are produced by the ShaderVariantListBuilder as intermediate assets. Provides configuration data about which shader variants should be used to create a ShaderVariantTreeAsset.

REMARK: These files will be named <Shader Name>.hashedvariantlist. Using the name and the subpath of this file We can figure out the name of the *.shader file.

AZStd::vector<AZStd::string> AZ::ShaderBuilder::ShaderBuilderUtility::IncludedFilesParser::ParseStringAndGetIncludedFiles ( AZStd::string_view  haystack) const

This static function was made public for testability purposes only. Parses the string @haystack, looking for "#include file" lines with a regular expression. Returns the list of relative paths as included by the file. REMARK: The algorithm may over prescribe what files to include because it doesn't discern between comments, etc. Also, a #include line may be protected by #ifdef macros but this algorithm doesn't care. Over prescribing is not a real problem, albeit potential waste in processing. Under prescribing would be a real problem.

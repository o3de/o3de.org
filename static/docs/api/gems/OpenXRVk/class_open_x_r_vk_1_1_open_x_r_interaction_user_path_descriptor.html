Detailed Description

A User Path descriptor describes the XrPath (as a string) that will be use to identify a Left or Right hand controller, or a Game pad controller.

Component Paths that are only supported under this User Path. This list can be empty. In case it is empty, it means that all component Paths are listed under the Interaction Profile Descriptor that owns this User Path Descriptor.

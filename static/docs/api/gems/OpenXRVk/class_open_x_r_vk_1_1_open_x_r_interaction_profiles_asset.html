Detailed Description

This asset defines a list of Interaction Profile Descriptors. The Engine only needs one of these assets, which is used to express all the different interaction profiles (aka XR Headset Devices) that are supported by OpenXR. Basically this asset contains data as listed here: https://registry.khronos.org/OpenXR/specs/1.0/html/xrspec.html#semantic-path-interaction-profiles

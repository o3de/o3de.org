Detailed Description

Special Case Traits specialization for the string_view type The C++ string_view class uses the StringType traits so that in ScriptCanvas string_view maps as a native string type in ScriptCanvas. This to allow the common C++ concepts of a "string" to be treated as a string in ScriptCanvas according to the ScriptCanvas spec
